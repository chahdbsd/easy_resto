## Easy Resto (Java — Clean Architecture)

Easy Resto est une application développée en **Java (IntelliJ)** en suivant les principes de la **Clean Architecture** :
le cœur métier est indépendant de la UI et de la base de données, avec une séparation claire des responsabilités.

### Architecture
- **Domain** : entités + règles métier
- **Use Cases** : cas d’usage 
- **Interface Adapters** : repositories / mappers / controllers
- **Infrastructure: accès DB + interface terminal

### Points clés
- Découplage fort 
- Repositories + injection de dépendances
- Code maintenable, testable, évolutif

### Technologies
Java, IntelliJ, Clean Architecture, OOP, persistance (JDBC), PostgresSQL
